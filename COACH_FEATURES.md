# CoachTrack — Coach Mode Features

## Dashboard
- View all your clients in one place with active/inactive status
- See each client's current weight, goal weight, and tracking activity at a glance
- Add new clients by searching their username
- Pull-to-refresh to sync latest data

## Client Profile
- Full overview of a client: goal, weight, activity level, diet preference
- Health notes panel: conditions, injuries, allergies, medications
- One-tap access to assign or modify any plan directly from the profile
- Recent progress log: weight, steps, mood, stress (last 7 entries)

## Diet Plan
- Set calorie and macro targets (protein, carbs, fat) for each meal slot: Breakfast, Lunch, Snack, Dinner
- Live daily totals update as you type
- Load and edit existing plans at any time

## Workout Plan
- Build multi-day rotation plans (e.g. 3-day, 5-day splits)
- Add, rename, or remove days freely
- Search and filter an exercise library by name or muscle group
- Add exercises to each day and set custom sets × reps
- **Per-exercise YouTube video support** — add a tutorial link from the assignment screen; coaches can watch the video in-app before assigning

## Lifestyle Plan
- Toggle standard habits on/off: Daily Steps, Water Intake, Sleep, Screen Time, Meditation, Sunlight, No Sugar, No Alcohol, Meal Timing
- Set a numeric target for each selected habit (e.g. 8000 steps/day)
- Add unlimited custom habits with your own name, target value, and unit
- Edit or delete custom habits inline

## Coach Tips
- Post short coaching notes or motivational tips to a client
- All tips are visible to the client and stored with timestamps
- View your full tip history per client
